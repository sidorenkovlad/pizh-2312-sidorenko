# Проект: Коллекция объектов DateTime

Этот проект содержит два основных класса:
- `DateTime` — класс для работы с датой и временем.
- `DateTimeCollection` — контейнер для хранения набора объектов `DateTime`.

---

## Класс `DateTime`

Класс предназначен для представления даты и времени, а также предоставляет методы для их обработки.

### Основные методы и свойства:

- `__init__(self, year, month, day, hour=0, minute=0, second=0)`  
  Конструктор. Создает объект `DateTime` с указанными компонентами даты и времени.

- `__str__(self)`  
  Возвращает строковое представление объекта в формате "YYYY-MM-DD HH:MM:SS".

- `from_string(cls, str_value)`  
  Создает объект `DateTime` из строки формата "YYYY-MM-DD HH:MM:SS".

- `save(self, filename)`  
  Сохраняет объект в файл (может быть реализовано как сохранение в JSON или другой формат).

- `load(cls, filename)`  
  Загружает объект из файла.

- `date(self)`  
  Возвращает строку с датой в формате "YYYY-MM-DD".

- `time(self)`  
  Возвращает строку с временем в формате "HH:MM:SS".

- `is_weekend(self)`  
  Проверяет, является ли дата выходным днем (суббота или воскресенье).

- `add_days(self, days)`  
  Добавляет указанное количество дней к дате.

- `difference_in_days(self, other)`  
  Вычисляет разницу в днях между текущим объектом и другим объектом `DateTime`.

---

## Класс `DateTimeCollection`

Класс представляет собой контейнер для хранения набора объектов `DateTime`. Обеспечивает управление коллекцией и её сохранение/загрузку.

### Основные методы и свойства:

- `_data` (поле)  
  Список объектов типа `DateTime`, хранящий коллекцию.

- `__init__(self, data=None)`  
  Инициализация коллекции. Можно передать список объектов `DateTime`.

- `__str__(self)`  
  Возвращает строковое представление всей коллекции (по одному объекту на строку).

- `__getitem__(self, index)`  
  Позволяет получать элементы по индексу или срезу.

- `add(self, value)`  
  Добавляет новый объект типа `DateTime` в коллекцию.

- `remove(self, index)`  
  Удаляет элемент по индексу.

- `save(self, filename)`  
  Сохраняет всю коллекцию в JSON-файл. Каждый объект сериализуется в словарь с датой и временем.

- `load(self, filename)`  
  Загружает коллекцию из JSON-файла. Восстанавливает объекты типа `DateTime`.

---

## Использование

```python
# Создание коллекции
collection = DateTimeCollection()

# Добавление объектов
collection.add(DateTime(2023, 10, 5))
collection.add(DateTime(2023, 12, 25))

# Вывод коллекции
print(collection)

# Удаление элемента
collection.remove(0)

# Сохранение в файл
collection.save('collection.json')

# Загрузка из файла
collection.load('collection.json')
