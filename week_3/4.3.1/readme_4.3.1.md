Roman Number Calculator
Описание проекта
Данный проект реализует набор классов для работы с римскими числами и их арифметическими операциями. В основе лежит класс Roman, который позволяет создавать объекты, представляющие римские числа, а также выполнять операции сложения.
Расширенная версия ExtendedRoman добавляет возможность преобразования числа в строковое представление. Класс Calculator служит для хранения коллекции римских чисел и выполнения их суммирования.
Этот проект предназначен для обучения работе с объектно-ориентированным программированием на Python, а также для практики реализации арифметических операций и наследования.

Основные возможности
Создание объектов римских чисел
Выполнение сложения двух римских чисел
Расширение функциональности через наследование
Хранение коллекции римских чисел
Суммирование всех добавленных чисел

Подробное описание классов
Класс Roman
Представляет римское число с возможностью сложения с другим объектом этого же класса.

Класс ExtendedRoman
Наследует от Roman, добавляя метод преобразования числа в строковое представление для удобства отображения.

Класс Calculator
Обеспечивает хранение коллекции объектов типа Roman и выполнение операций суммирования всех добавленных чисел.
