# Проект "Система заказа пиццы"

Этот проект представляет собой простую объектно-ориентированную систему для моделирования процесса заказа пиццы. Он включает в себя основные классы, описывающие виды пицц и механизм оформления заказа, что позволяет легко расширять функциональность и добавлять новые виды пицц или методы обработки заказов.

---

## Описание проекта

Данный проект реализует концепцию системы заказа пиццы с помощью языка программирования Python. В основе лежит использование принципов объектно-ориентированного программирования: инкапсуляции, наследования и полиморфизма.

Ключевые компоненты системы:

- **Класс `Pizza`** — абстрактный базовый класс, содержащий общие свойства и методы для всех видов пицц.
- **Классы конкретных видов пицц** (например, `PepperoniPizza`, `BBQPizza`) — наследуются от базового класса `Pizza` и реализуют специфические особенности.
- **Класс `Order`** — управляет списком заказанных пицц, позволяет добавлять новые позиции, подсчитывать итоговую сумму и выводить информацию о заказе.

Данная структура позволяет легко добавлять новые виды пицц или расширять функциональность системы без изменения существующего кода.

---

## Основные возможности

- Создание различных видов пицц с уникальными характеристиками.
- Формирование заказа с возможностью добавления нескольких позиций.
- Подсчет общей стоимости заказа.
- Вывод информации о заказе в удобочитаемом виде.
- Простая расширяемость системы для поддержки новых видов пицц или дополнительных функций.

---

## Структура проекта

Проект состоит из следующих компонентов:

- **Модуль с классами**: содержит определения классов `Pizza`, `PepperoniPizza`, `BBQPizza`, `Order`.
- **Основной скрипт** (например, `main.py`) — точка входа в программу, где создаются объекты и осуществляется взаимодействие пользователя с системой.

---

## Как использовать

1. **Установка Python 3.x**

   Для запуска системы потребуется установленная версия Python 3.x. Скачать её можно с официального сайта [python.org](https://www.python.org/).

2. **Загрузка проекта**

   Скачайте или склонируйте репозиторий с исходным кодом на ваш компьютер.

3. **Запуск программы**

   В командной строке перейдите в директорию проекта и запустите основной скрипт (например, `main.py`). Перед этим убедитесь, что все необходимые файлы находятся в месте.

4. **Расширение функциональности**

   Для добавления новых видов пицц создайте новые классы, наследующиеся от базового класса `Pizza`, и реализуйте необходимые методы.
