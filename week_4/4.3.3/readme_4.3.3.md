# Вкладной калькулятор на Python

Этот проект представляет собой простое консольное приложение на Python для расчёта доходности различных видов вкладов. Пользователь может выбрать тип вклада, ввести параметры и получить итоговую сумму и прибыль.

---

## Описание проекта

Данный скрипт реализует систему моделирования вкладов с разными условиями:

- **Фиксированный вклад** — ставка процента фиксирована на весь срок.
- **Вклад с бонусом** — при достижении определённого порога суммы начисляется бонус.
- **Вклад с капитализацией** — проценты добавляются к основной сумме, увеличивая доходность.

Каждый тип вклада реализован как отдельный класс, наследующий от базового класса `Deposit`. Взаимодействие с пользователем осуществляется через командную строку.

---

## Структура классов и их работа

### Базовый класс: `Deposit`

- **Описание:**  
  Этот класс служит основой для всех видов вкладов. Он содержит общие свойства и методы, такие как:
  - `amount` — сумма вклада
  - `term_months` — срок вклада в месяцах
  - `interest_rate` — годовая ставка процента
  - Методы для расчёта итоговой суммы (`calculate_final_amount`) и прибыли (`calculate_profit`).

- **Работа:**  
  В базовом классе реализованы общие механизмы расчёта процентов по формуле простого или сложного процента (зависит от реализации). От этого класса наследуются все конкретные виды вкладов, которые могут переопределять методы для специфического поведения.

---

### Класс: `FixedDeposit` (Фиксированный вклад)

- **Описание:**  
  Наследует от `Deposit`.  
  Использует стандартный расчет процентов без дополнительных условий.

- **Особенности:**  
  Расчет итоговой суммы основан на фиксированной ставке за весь срок.  
  Метод `calculate_final_amount()` возвращает сумму с начисленными процентами за весь период.

---

### Класс: `BonusDeposit` (Вклад с бонусом)

- **Описание:**  
  Наследует от `Deposit`.  
  Добавляет возможность начисления бонуса при достижении определенного порога суммы.

- **Особенности:**  
  В методе `calculate_final_amount()` учитывается бонус, если сумма вклада превышает заданный порог.  
  Например, если сумма превышает порог, к итоговой сумме добавляется бонус в виде процента или фиксированной суммы.

---

### Класс: `CapitalizedDeposit` (Вклад с капитализацией)

- **Описание:**  
  Наследует от `Deposit`.  
  Реализует капитализацию процентов — начисленные проценты добавляются к основной сумме и участвуют в следующем периоде.

- **Особенности:**  
  В методе `calculate_final_amount()` реализована капитализация по частям (например, ежемесячно). Это увеличивает итоговую сумму по сравнению с простым процентом.

---

## Как работают классы вместе

1. При запуске программы создается объект выбранного типа вклада (`FixedDeposit`, `BonusDeposit`, или `CapitalizedDeposit`) с введенными пользователем параметрами.
2. Для каждого объекта вызываются методы:
   - `calculate_final_amount()` — чтобы получить итоговую сумму по окончании срока.
   - `calculate_profit()` — чтобы определить прибыль (разницу между итоговой суммой и первоначальной).
3. Результаты выводятся пользователю.
