# Класс DateTime на Python

Этот проект содержит реализацию класса `DateTime`, который позволяет работать с датами и временем, выполнять арифметические операции, сохранять и загружать объекты из файла, а также предоставлять дополнительные методы для работы с датами.

---

## Описание

Класс `DateTime` реализован на Python и использует встроенный модуль `datetime`. Он включает в себя:

- Создание объектов с заданной датой и временем
- Представление даты и времени в виде строки
- Арифметические операции сложения и вычитания
- Сохранение объекта в файл и загрузка из файла
- Получение текущей даты и времени
- Проверка, является ли дата выходным днем (выходным)
- Добавление дней к дате
- Вычисление разницы в днях между двумя датами

---

## Описание классов

### Класс `DateTime`

Представляет дату и время с дополнительными методами для работы с ними.

#### Атрибуты:
- `_dt` — внутренний объект `datetime` из стандартной библиотеки Python, хранящий дату и время.

#### Методы:

- `__init__(self, year, month, day, hour=0, minute=0, second=0)`  
  Конструктор. Создает объект `DateTime` с указанными компонентами даты и времени.

- `__str__(self)`  
  Возвращает строковое представление объекта в формате `'YYYY-MM-DD HH:MM:SS'`.

- `__add__(self, other)`  
  Выполняет сложение: если `other` — `timedelta`, возвращает новый объект `DateTime`; если `other` — другой `DateTime`, возвращает разницу (`timedelta`).

- `__sub__(self, other)`  
  Выполняет вычитание: если `other` — `timedelta`, возвращает новый объект `DateTime`; если `other` — другой `DateTime`, возвращает разницу (`timedelta`).

- `from_string(cls, str_value)`  
  Создает объект из строки формата `'YYYY-MM-DD HH:MM:SS'`.

- `save(self, filename)`  
  Сохраняет текущий объект в файл.

- `load(cls, filename)`  
  Загружает объект из файла.

- `date(self)`  
  Возвращает строку с датой `'YYYY-MM-DD'`.

- `time(self)`  
  Возвращает строку с временем `'HH:MM:SS'`.

- `is_weekend(self)`  
  Проверяет, является ли дата выходным днем (суббота или воскресенье).

- `add_days(self, days)`  
  Возвращает новый объект с добавленными днями.

- `difference_in_days(self, other)`  
  Возвращает количество дней между текущим объектом и другим объектом типа `DateTime`.
