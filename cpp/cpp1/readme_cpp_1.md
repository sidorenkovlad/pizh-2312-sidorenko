# Телефонный справочник

Это простое консольное приложение на C++, реализующее базовую структуру телефонного справочника с использованием двух классов: `Spravochnik` и `Kontakt`.

---

## Описание классов

### Класс `Kontakt`

- **Назначение:** Представляет отдельный контакт в справочнике.
- **Данные:**
  - `name` — имя контакта.
  - `phone` — номер телефона.
  - `address` — адрес.
- **Методы:**
  - Конструктор для инициализации данных.
  - `display()` — выводит информацию о контакте на экран.
  - Геттеры для получения данных (по необходимости).

---

### Класс `Spravochnik`

- **Назначение:** Представляет телефонный справочник, содержащий список контактов.
- **Данные:**
  - `title` — название справочника.
  - `ownerName` — имя владельца справочника.
  - `contacts` — вектор объектов класса `Kontakt`.
- **Методы:**
  - Конструктор для задания названия и имени владельца.
  - `addContact(const Kontakt&)` — добавляет новый контакт в справочник.
  - `display()` — выводит информацию о справочнике и всех его контактах.
