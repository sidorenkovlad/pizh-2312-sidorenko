Описание проекта "Касса"
Данный проект представляет собой программное обеспечение для работы с кассовым аппаратом, реализующее основные функции учета продаж, хранения данных о товарах и операциях. Проект выполнен на языке C++ и использует объектно-ориентированный подход.

Основные классы
1. Класс Product (Товар)
Описание:
Представляет товар, продаваемый в магазине.

Поля:

std::string name — название товара
double price — цена за единицу товара
int quantity — количество товара на складе
Методы:

Конструкторы: по умолчанию и с параметрами
Геттеры и сеттеры для каждого поля
void display() — вывод информации о товаре
void saveToFile(const std::string& filename) — сохранение данных в файл
void loadFromFile(const std::string& filename) — загрузка данных из файла
2. Класс CashRegister (Касса)
Описание:
Обеспечивает работу с операциями продажи товаров.

Поля:

std::vector<Product> products — список товаров в магазине или текущая смена
double totalSum — итоговая сумма продаж
Методы:

Добавление товара (addProduct)
Удаление товара (removeProduct)
Обновление количества или цены товара (updateProduct)
Просмотр списка товаров (displayProducts)
Проведение продажи (sellProduct) — уменьшает количество товара и увеличивает итоговую сумму
Сохранение и загрузка данных о товарах из файла
Основные функции
Ввод и вывод данных о товарах через операторы << и >>.
Работа с файлами для сохранения/загрузки данных.
Обработка исключений при ошибках ввода, чтения/записи файлов, некорректных данных.
Использование контейнеров (std::vector) для хранения списка товаров.
Использование итераторов для обхода коллекций.
Использование
Создайте объект кассы (CashRegister).
Добавьте товары в кассу или загрузите их из файла.
Просмотрите список товаров.
Выполните операции продажи, обновления или удаления товаров.
