# Программа управления товарами на C++

Данная программа реализует работу с коллекцией товаров (`Product`) с возможностью сортировки, поиска и перемещения элементов между контейнерами.

---

## Структура программы

### Класс `Product`

Класс `Product` представляет товар и содержит следующие поля:
- `name` — название товара (строка)
- `price` — цена товара (число с плавающей точкой)
- `quantity` — количество товара (целое число)

**Методы класса:**
- Конструкторы:
  - По умолчанию (`Product()`)
  - Инициализации с параметрами (`Product(const std::string&, double, int)`)
- Геттеры:
  - `getName()` — возвращает название
  - `getPrice()` — возвращает цену
  - `getQuantity()` — возвращает количество
- Сеттеры:
  - `setName()`, `setPrice()`, `setQuantity()` — позволяют изменить соответствующие поля
- `display()` — выводит информацию о товаре в консоль

---

## Основные функции программы

- **Создание контейнера товаров:** Вектор `std::vector<Product>` содержит список товаров.
- **Вывод содержимого:** Функция `displayProducts()` выводит все товары из контейнера.
- **Сортировка:**
  - По убыванию цены (`std::sort` с компаратором)
  - По возрастанию цены (после перемещения элементов)
- **Поиск:** Использование `std::find_if()` для поиска товара по условию (например, цена ниже определенного значения).
- **Перемещение элементов:** Товары, удовлетворяющие условию, копируются в другой контейнер, а затем удаляются из исходного.
  
---

## Использование

1. В программе создается исходный список товаров.
2. Выполняется сортировка по убыванию цены.
3. Производится поиск товара по условию.
4. Товары, соответствующие условию, перемещаются в отдельный контейнер.
5. Оба контейнера сортируются по возрастанию цены.
6. Итоговые списки выводятся на экран.
