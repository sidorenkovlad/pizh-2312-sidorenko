Описание программы и классов
Данная программа демонстрирует обработку исключительных ситуаций в C++, включая стандартные типы исключений, пользовательские классы ошибок и обработку исключений на разных уровнях (локальном и верхнем).

Основные классы

1. MyLogicError
Наследует: std::logic_error
Описание: Пользовательский класс ошибки, предназначенный для представления логических ошибок в программе.
Конструктор:
cpp
Копировать код
explicit MyLogicError(const std::string& message);
Создает объект ошибки с заданным сообщением.

2. MyRuntimeError
Наследует: std::runtime_error
Описание: Пользовательский класс ошибки, предназначенный для ошибок выполнения.
Конструктор:
cpp
Копировать код
explicit MyRuntimeError(const std::string& message);
Создает объект ошибки с заданным сообщением.

Основные функции
generateExceptions(int choice)

Описание: Генерирует исключения различных типов в зависимости от переданного параметра choice.

Используемые типы исключений:
int (например, throw 42;)
std::string (например, throw std::string("Ошибка строки");)

Стандартные исключения (std::invalid_argument, std::length_error, std::out_of_range)

Пользовательские (MyLogicError, MyRuntimeError)
processWithLocalCatch(int choice)

Описание: Вызов функции generateExceptions() внутри блока try-catch для локальной обработки исключений.
Обрабатывает исключения типа int, std::string, а также пользовательские и стандартные исключения.

Обработка исключений
В функции main() реализована верхняя обработка всех возможных исключений, вызванных функциями внутри нее.
Используются блоки try-catch для перехвата конкретных типов и универсальный перехват через catch(...).

Итог
Данная программа служит учебным примером по работе с исключениями в C++, показывая:

Генерацию различных типов исключений
Локальную обработку
Обработку на верхнем уровне
Создание собственных классов ошибок
Использование стандартных исключений
