# Телефонный справочник

Это консольное приложение на C++, реализующее телефонный справочник с использованием классов. В программе используются два основных класса: `Kontakt` и `Spravochnik`.

---

## Описание классов

### Класс `Kontakt`

- **Назначение:** Представляет отдельный контакт в телефонном справочнике.
- **Данные:**
  - `name` — имя контакта.
  - `phone` — номер телефона.
  - `address` — адрес.
- **Конструкторы:**
  - Без параметров — создает пустой контакт, выводит сообщение о вызове.
  - С параметрами — инициализирует все поля, выводит сообщение о вызове.
  - Конструктор копирования — создает копию другого контакта, выводит сообщение.
- **Деструктор:** выводит сообщение при уничтожении объекта.
- **Методы:**
  - `display()` — выводит информацию о контакте.

---

### Класс `Spravochnik`

- **Назначение:** Представляет телефонный справочник, содержащий список контактов.
- **Данные:**
  - `title` — название справочника.
  - `ownerName` — имя владельца.
  - `contacts` — вектор объектов класса `Kontakt`.
- **Конструкторы:**
  - Без параметров — создает пустой справочник, выводит сообщение.
  - С параметрами — задает название и владельца, выводит сообщение.
  - Конструктор копирования — копирует другой справочник, выводит сообщение.
- **Деструктор:** выводит сообщение при уничтожении объекта.
- **Методы:**
  - `addContact(const Kontakt&)` — добавляет контакт в список.
  - `display()` — выводит информацию о справочнике и всех контактах.

---

## Использование

В основном файле создаются объекты классов разными способами (без параметров, с параметрами, копированием), демонстрируется их создание и уничтожение. Также реализованы методы для добавления контактов и отображения содержимого.
