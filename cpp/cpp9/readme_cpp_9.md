Описание проекта
Данный проект представляет собой расширение программы, разработанной согласно варианту лабораторной работы №8. В проекте реализованы базовые классы с возможностью ввода, вывода, сохранения и загрузки данных из файла, а также обработка исключений.

Основные классы
1. Класс Person
Описание:
Базовый класс, представляющий человека с именем и возрастом.

Поля:

std::string name — имя человека
int age — возраст человека
Методы:

Конструкторы: по умолчанию и с параметрами
saveToFile(const std::string& filename) — сохраняет данные объекта в указанный файл
loadFromFile(const std::string& filename) — загружает данные из файла в объект
Друзья-функции:
operator<< — выводит данные объекта на экран
operator>> — считывает данные с клавиатуры или из потока
Операторы
Глобальные операторы << и >>
operator<<: выводит содержимое объекта Person в поток (например, на экран). Формат: "Имя: <name>, Возраст: <age>".

operator>>: считывает данные для объекта Person из потока (например, с клавиатуры). Запрашиваются имя и возраст.

Методы сохранения и загрузки
saveToFile(const std::string& filename): сохраняет текущие данные объекта в файл. В файле записываются имя и возраст по строкам.

loadFromFile(const std::string& filename): загружает данные из файла в объект. Ожидается, что файл содержит имя и возраст в указанном формате.

Исключения
В проекте реализована обработка различных исключений:

Стандартные исключения (std::invalid_argument, std::length_error, std::out_of_range)
Пользовательские исключения (MyLogicError, MyRuntimeError) — наследуются от стандартных классов ошибок и предназначены для обработки логических ошибок и ошибок выполнения соответственно.
Также реализована функция генерации исключений для демонстрации обработки.

Использование
Ввод данных пользователя осуществляется через оператор >>.
Объект можно сохранить в файл командой saveToFile.
Объект можно загрузить из файла командой loadFromFile.
Для вывода данных используется оператор <<.
Обработка исключений осуществляется через блоки try-catch.
Итог
Данный проект демонстрирует работу с классами, операторами ввода/вывода, файлами и обработкой исключений на языке C++. Он может быть использован как основа для дальнейших расширений или учебных целей по изучению ООП и работы с файлами.
