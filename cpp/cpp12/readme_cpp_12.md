Описание проекта
Данный проект представляет собой программу для работы с коллекцией объектов класса Product. В программе реализованы операции сортировки, поиска и перемещения элементов контейнера с использованием алгоритмов стандартной библиотеки C++ и лямбда-функций.

Класс Product
Назначение
Класс Product моделирует товар с характеристиками: название, цена и количество.

Поля
std::string name — название товара.
double price — цена товара.
int quantity — количество товара.

Методы
Конструкторы
Product() — конструктор по умолчанию, инициализирует поля пустыми или нулевыми значениями.
Product(const std::string& n, double p, int q) — параметризированный конструктор для задания всех полей.

Геттеры
std::string getName() const — возвращает название товара.
double getPrice() const — возвращает цену товара.
int getQuantity() const — возвращает количество товара.

Сеттеры
void setName(const std::string& n) — задает название товара.
void setPrice(double p) — задает цену товара.
void setQuantity(int q) — задает количество товара.

Другие методы
void display() const — выводит информацию о товаре в консоль.

Основные функции программы
Создание исходного контейнера с объектами класса Product.
Сортировка контейнера по цене (по убыванию и возрастанию) с помощью лямбда-функций.
Поиск элемента по условию (например, цена ниже заданного значения).
Перемещение элементов, удовлетворяющих условию, в отдельный контейнер.
Вывод содержимого контейнеров.

Используемые технологии
C++11 и выше (использование лямбда-функций, алгоритмов стандартной библиотеки).
STL-контейнеры (std::vector).

Заключение
Данный проект демонстрирует работу с классами, алгоритмами STL и лямбда-функциями для обработки коллекций объектов. Использование лямбда-функций делает код более компактным и удобным для модификации условий обработки данных.
