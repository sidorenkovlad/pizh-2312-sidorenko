Worker Class
Этот проект содержит класс Worker, который моделирует сотрудника компании. Класс позволяет хранить информацию о сотруднике и выводить ее с правильным склонением слова "год" в зависимости от стажа.

Описание
Класс Worker имеет следующие свойства:
name — имя сотрудника
position — должность
experience — стаж работы в годах
Метод print_info() выводит информацию о сотруднике, корректно склоняя слово "год" в зависимости от количества лет опыта.

Использование
Пример создания объектов и вывода информации:

worker1 = Worker("Алексей", "Программист", 17)
worker1.print_info()

worker2 = Worker("Анна", "Маркетолог", 2)
worker2.print_info()

worker3 = Worker("Дмитрий", "Аналитик", 1)
worker3.print_info()

Вывод:
Имя: Алексей
Должность: Программист
Стаж: 17 лет

Имя: Анна
Должность: Маркетолог
Стаж: 2 года

Имя: Дмитрий
Должность: Аналитик
Стаж: 1 год
